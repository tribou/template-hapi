Starting template for Hapi.js apps

#### Quick Start

```
git clone https://github.com/tribou/template-hapi.git
cd template-hapi
npm install
npm start
```

#### Testing

Testing is performed using the [lab](https://npm.org/packages/lab) module.

Code coverage requirement is 100%.

```
npm test
```
